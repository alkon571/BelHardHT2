проект в IDE
3. Инициализировать git репозиторий (git init)
4. Создать коммит, который содержит gitignore
5. Создать ветку (1) <ваша_фамилия> из ветки master
6. Создать ветку (2) <ваше_имя> из ветки master
7. В ветке (1) создать файл и сделать commit
8. В ветке (2) создать файл и сделать commit
9. Сделать merge ветки (1) в ветку (2)
10. Изменить файл из пункта 7 и сделать commit
11. Перейти в ветку (2) и сделать cherry-pick коммита из пункта 10
12. Из ветки (1) сделать ветку (3) <ваше_отчество>
13. Сделать rebase ветки (3) от ветки (2)
14. В ветке (2) добавить новый файл и сделать коммит
15. В ветке (2) сделать revert на предыдущий коммит
16. В ветке (1) сделать 2 коммита
17. В ветке (1) сделать reset на предыдущий коммит
18. Запушить все ветки на Github
19. На GitHub добавить файл README.md в проект в ветку master с текстом «# Welcome Git»
20. Сделать pull ветки master
21. Сделать pull request ветки (2) в ветку master
